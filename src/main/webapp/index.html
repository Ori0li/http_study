http 수업을 위해 다이나믹 웹 프로젝트 서버

  • 리소스 : 참조할 수 있는 자료
  • 웹 상의 리소스 : 인터넷에서 참조할 수 있는 자료(브라우저 url 주소로 참조할 수 있는 것, Hyper Text Transfer Protocol(http) 통신으로 참조 할 수 있는 것)
  • 정적 리소스 : 참조를 해도 변화가 없는 자료 (html,css,javascript,img,mp3,mp4....) => frontend : 화면을 꾸미거나 브라우저에서 동작
  • 동적 리소스 : 참조할때마다 데이터가 바뀌는 자료 (java,python,php,asp,node.....) => backend : data를 처리하거나 서버에서 실행
